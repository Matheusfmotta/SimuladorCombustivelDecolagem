Este é um projeto desenvolvido em Java com o objetivo de calcular o consumo de combustível de uma aeronave bimotor durante sua decolagem.
O sistema utiliza variáveis físicas reais que influenciam esse consumo,tais como (Atrito na pista, Densidade do ar, Massa total da aeronave, Peso da aeronave).

Além do viés técnico aeronáutico, este projeto pessoal tem como objetivo acadêmico colocar em prática conceitos fundamentais de Programação Orientada a Objetos (POO) e sintaxe Java.

A versão atual do projeto utiliza como base a aeronave Boeing 737-800 equipada com motores CFM56-7B27. Esta aeronave foi escolhida por dois motivos principais:
É amplamente operacional nos dias atuais e existe um grande nível de disponibilidade de informações técnicas e especificações comerciais sobre ela.

Futuramente, o objetivo é expandir o sistema para oferecer uma interface onde o usuário possa fornecer dados de qualquer aeronave. Isso permitirá:
- Calcular o consumo de combustível personalizado durante a decolagem;
- Obter dados comparativos entre diferentes modelos;
- Gerar conclusões analíticas sobre eficiência.
